API Verdulería - Flask + Swagger

Este proyecto consiste en una API REST desarrollada con Flask y documentada con Swagger utilizando Flasgger.

La aplicación simula una pequeña verdulería con productos y un carrito de compras manejado en memoria.

Tecnologías utilizadas:
Python
Flask
Flasgger (Swagger)

Funcionalidades implementadas
Obtener productos
